# PKMN

A terminal-style Pokédex and defense type effectiveness calculator built with Vue 3.

## Features

- Search Pokémon by name
- Displays the Pokémon's type(s) with color-coded badges
- Calculates full defense type effectiveness (4×, 2×, 1×, 0.5×, 0.25×, 0×) accounting for dual types

## Tech Stack

- [Vue 3](https://vuejs.org/) + TypeScript
- [Vite](https://vite.dev/)
- [Tailwind CSS](https://tailwindcss.com/)

## Getting Started

```bash
npm install
npm run dev
```

Open `http://localhost:5173` and type a Pokémon name at the prompt (e.g. `charizard`).
